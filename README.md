# Linear Regression – House Price Prediction

## Overview
This project implements **Linear Regression** to predict house prices using the
House Price Prediction dataset. The goal is to understand regression modeling,
evaluate performance, and interpret coefficients.

## Steps Performed
1. Loaded dataset
2. Handled missing values
3. Selected useful features
4. Split data into train and test sets
5. Trained Linear Regression model
6. Evaluated performance (MAE, MSE, RMSE, R²)
7. Visualized Actual vs Predicted values
8. Interpreted coefficients

## Evaluation Metrics
- MAE
- MSE
- RMSE
- R² Score
